# 🏦 Bank Account Management System (Python)

A simple **Bank Account Management System** built using Python.  
This project simulates basic banking operations like login, deposit, withdrawal, balance checking, and transaction history.

## 📌 About The Project

This program uses **Object-Oriented Programming (OOP)** concepts to create a simple banking system.

Users can:
- Login using a PIN
- Deposit money
- Withdraw money
- Check account balance
- View transaction history

## ✨ Features

✅ PIN authentication  
✅ Deposit functionality  
✅ Withdrawal functionality  
✅ Balance enquiry  
✅ Transaction history with date & time  
✅ Menu-driven system  
✅ OOP-based structure  

## 🛠️ Technologies Used

- Python 3.x
- datetime module
